### 使用方法

1. clone本项目并进入项目目录

    ```bash
    git clone https://github.com/laoshuikaixue/Campus-Radio-Station-Audio-Processing-System.git
    cd Campus-Radio-Station-Audio-Processing-System
    ```

2. 安装依赖

    ```bash
    python -m pip install -r ./requirements.txt
    ```

3. 启动

    ```bash
    python main.py
    ```
